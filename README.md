# VerticalSeekbar

# Install Dependency

If you haven't already, add the jitpack repository to the root build.gradle file.

# 
    allprojects {
       repositories {
           ...
           maven { url 'https://jitpack.io' }
       }
    }

 # Add the dependency:

    dependencies {
	        implementation 'com.github.viveksaini75:VerticalSeekbar:1.0.1'
	}
